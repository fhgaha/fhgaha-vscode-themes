## Installation
Install by name like any other vscode extension or

In vscode F1 -> Preferences: Color Theme -> Enter -> fhgaha -> "Browse Other Themes" line appears -> Enter -> fhgaha -> Enter ->
"This will install this extension. Continue?" -> OK
